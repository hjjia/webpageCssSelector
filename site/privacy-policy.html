CSS Master 隐私政策

生效日期：2026-01-05 · English

CSS Master 是一个用于分析当前页面 CSS 样式的浏览器扩展。我们高度重视您的隐私与数据安全，并承诺遵循以下原则：

  • 纯本地运行：所有处理在您的浏览器内完成。
  • 不收集数据：不收集、存储或传输任何用户数据。
  • 不联网：除浏览器自身加载页面资源外，扩展不主动发起网络请求。

数据收集与使用

我们不收集任何可识别个人身份的信息，也不记录、分析或共享您的使用行为。扩展仅在本地读取当前页面的样式信息以实现功能展示。

  • 不收集个人信息（如姓名、邮箱、设备标识）。
  • 不跟踪行为（不使用分析、埋点或追踪脚本）。
  • 不上传数据（不向任何服务器传输页面内容或分析结果）。
  • 不持久化存储（关闭页面或扩展后不保留数据）。

权限与适用范围

为实现功能，扩展可能读取当前页面的 DOM/CSS 信息，用于本地计算与展示。这些信息仅在您的设备本地处理，不会跨站共享或外传。

  • 不访问浏览器历史、Cookies、文件系统等敏感数据。
  • 不修改网站内容，除非为展示样式分析所需的临时本地渲染。
  • 不跨域访问其他网站数据。

第三方服务

扩展不集成任何第三方分析、广告或统计服务，也不使用第三方 SDK 收集数据。

儿童隐私

扩展不面向儿童，不收集任何儿童个人信息。

政策更新

如本隐私政策发生变更，我们将在本页面发布更新版本。更新后继续使用即表示您同意新的政策内容。

联系我们

如对本隐私政策有任何问题或反馈，请通过扩展的发布页面与我们取得联系。

CSS Master 是一个纯本地工具，旨在帮助您更好地理解当前页面的 CSS 样式。我们不会也无法访问或传输您的任何个人数据。
